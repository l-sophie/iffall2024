Recipe Page

This page features a pumpkin cinnamon roll recipe from Amitious Kitchen's Monique Volz. I included important information regarding the recipe with clear visual hierarchy and accompanying instructional images to make the website easy for anyone to follow.

All Kinds of Spicy — Designer

With the creative direction of Yuechen Zhang, I designed an informational, single-page website about spicy food, including information on how the sensation of spice works, popular hot sauces, recipe ideas, and more. I used a neutral/muted color palette to allow the red and brighter images to pop and add a playful element to the website, which went well with the fun serif font. This website is responsive to different screen sizes.

Spider-Man: Across the Spiderverse — Creative Director

I directed Analise Black to design an informational, multi-page website about the movie Spider-Man: Across the Spiderverse. We utilized aspects of the movie to inspire the color palette (dark backdrop with pops of bright, futuristic, cyber-punk colors), font (graffiti), and design (comic-style text boxes) of the website. It highlights the plot, characters, production, and revolutionary animation of the movie.
